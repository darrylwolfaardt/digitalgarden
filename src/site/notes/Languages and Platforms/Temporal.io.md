## What is it?

Temporal is an open-source durable execution platform that makes building reliable distributed applications straightforward. It handles the complexity of failures, retries, and state management automatically, allowing developers to write workflows as simple code while the platform guarantees execution completion even through crashes, restarts, or network outages.
